个人blog创建方法

安装nodejs和git

首先进入https://nodejs.org/按照电脑系统下载对应.msi文件，双击文件安装，选择安装路径的时候建议在空间较大的盘内新建一个命名为nodejs的文件夹专门用来安装,右键点击该文件，点击安全，点击编辑，点击权限，全选 ，win+D输入cmd输入nvm -v 检查有无版本号，有版本号即代表安装成功，安装成功后输入npm config set registry https://registry.npmmirror.com 切换下载镜像源，国内没有挂梯子下载会很慢，换镜像源下载速度会提升许多，cmd内输入npm config get registry检查安装的镜像源是否成功

在nodejs文件夹内新建node_cache和node_global两个新文件夹，在路径处全选输入cmd回车，输入npm config set prefix “盘名:\nodejs\node_global” npm config set cache “盘名:\nodejs\node_cache”

用户变量下的Path点击进去将npm修改成 盘名:\nodejs\node_global 点击确定，在系统变量下新建NODE_PATH,变量值为 盘名:\nodejs\node_global\moudles，在系统变量的Path新建 盘名:\nodejs\node_global,这样后续下载的模块会自动导入指定路径

在cmd内输入npm install express -g，下载成功 盘名:\nodejs\node_global\moudles文件下会新增一个express文件夹

安装git工具，提前创建github账号，没有梯子就输入kkgithub.com进入镜像

git下载网址：https://git-scm.com/ 选择最新安装包，下载好后双击下一步，如上提前新建好一个名为git的文件夹，安装在该文件夹内，第五步有“Don’t create a start menu folder”要打勾，然后下一步就行。

cmd输入git -v查看到版本信息即是安装成功，在一个较大盘内新建名为hexo的文件夹，git bash here打开窗口
输入npm install -g hexo-cli安装hexo框架，输入hexo -v出现版本信息说明安装成功
在新建的文件夹内右键git bash here输入hexo init ”项目名“ 初始化项目，项目名尽量起的与后续工作相关

cd 项目名 进入这个文件夹 npm install 暂时只需要关注初始化成功后出现的source文件内_post文件内的博客md文件（存文章），theme文件夹（存博客主题），_config.yml文件（博客配置信息）
theme文件主要是用来下载hexo主题的，可以去hexo官网寻找喜欢的主题git clone代码到这个文件夹
最后把_config.yml文件内的theme和最下面.deploy文件信息配置好theme这里的名字和下载的名字必须一模一样，_post文件夹里面的两个文件名也是必须一模一样，建议复制粘贴，不然hexo d的时候会出error

创建好github账号就创建一个个人仓库，起名为用户名.github.io方便记，将c盘内.ssh文件生成的ssh添加到github，再将hexo部署到github，git bash here 输入npm install hexo-deployer-git –save

hexo部署方法为打开_congif.yml文件拉到最下面.deploy输入:
type: git
repo: https://github.com/用户名/用户名.github.io.git //和上面创建的仓库名一致
branch: master //或者main，看你提交文件左上角是哪一个

搞定好就在git窗口里hexo cl && hexo g &&hexo s 在浏览器输入localhost:4000查看本地博客页面情况，最后hexo g && hexo d部署，git pull 远程同步一下
最后直接在浏览器输入https://用户名.github.io观察博客情况
